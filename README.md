# glonetzErpFrontend

Interface frontend Glonetz ERP (Next.js), issue du projet [v0](https://v0.app).

## Démarrage

```bash
npm install
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans le navigateur.
